# postman

## 1. 安装

- [官网](https://www.postman.com/downloads/)下载安装包；

## 2. 配置

1. 默认安装后，使用账号密码登录；
2. 同步请求记录；

## 3. 使用

- [官方文档](https://learning.postman.com/docs/introduction/overview/)；

### 3.1. 基础

- Collection，建议按业务、工程分组；
- http请求调试：
  - post；
  - get；
  - update；
  - delete；
- 调用或编写接口前后先测试接口正确性；

### 3.2. 进阶

- 小范围压测；
- 变量赋值、传递；
- grpc；
- 监听服务；
